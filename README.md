# Life Expectancy Data Analyzer

## Description

This project is a Python program that analyzes life expectancy data from different countries.

The program reads information from a CSV file and allows users to:
- Find the overall highest life expectancy
- Find the overall lowest life expectancy
- Calculate the average life expectancy for a selected year
- Find the country with the highest and lowest life expectancy for a specific year
- Find statistics for a specific country
- Identify the largest life expectancy drop over a 5-year period

## Features

- User input validation
- Handles invalid years and countries
- Calculates statistics automatically
- Allows multiple searches
- Uses CSV data processing
